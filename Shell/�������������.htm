偶又罗里罗嗦写了一堆， :oops: 越要精简越多 :oops: 凑合用吧。
  1. #!/bin/sh
  2. # countdate #计算两个日期之间有多少天
  3. a1=`echo $1|cut -c1-4`
  4. a2=`echo $1|cut -c5-6|sed 's/^0//'`
  5. a3=`echo $1|cut -c7-8|sed 's/^0//'`
  6. b1=`echo $2|cut -c1-4`
  7. b2=`echo $2|cut -c5-6|sed 's/^0//'`
  8. b3=`echo $2|cut -c7-8|sed 's/^0//'`
  9. sum()   
  10. {
  11. case $1 in
  12. 1|3|5|7|8|10|12) ff=31;;
  13. 2) ff=`cal $1 $2|sed '/^$/d'|sed -n '$p'|awk '{print $NF}'`;;
  14. 4|6|8|9|11) ff=30;;
  15. esac
  16. }
  17. if [ $a2 -ne 1 ];then
  18.   p=1;a2=`expr $a2 - 1`;beg=0
  19.   while [ $p -le $a2 ];do sum $p $a1;beg=`expr $beg + $ff`;p=`expr $p + 1`;done
  20. else
  21.   beg=0
  22. fi
  23. if [ $b2 -ne 12 ];then
  24.   end=0
  25.   while [ $b2 -le 12 ];do sum $b2 $b1;end=`expr $end + $ff`;b2=`expr $b2 + 1`;done
  26. else
  27.   end=31
  28. fi
  29. a11=$a1;mm=0
  30. while [ $a11 -le $b1 ];do
  31.   rr=`cal 2 $a11|sed '/^$/d'|sed -n '$p'|awk '{print $NF}'`
  32.    if [ "$rr" = "28" ];then
  33.      mm=`expr $mm + 365`
  34.    else
  35.      mm=`expr $mm + 366`
  36.    fi
  37.   a11=`expr $a11 + 1`
  38. done
  39. echo $mm-$beg-$a3-$end+$b3|bc
复制代码


SCO5.05下测试情况：
  1. sh countdate 20010202 20010221
  2. 19
  3. sh countdate 20040226 20040302
  4. 5
  5. sh countdate 20041226 20050103
  6. 8

  8. 当然天数越多效率就越低了，偶测了个100年有多少天用了约3秒多 :oops:
